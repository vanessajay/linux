Dica: A ferramenta para realizar o desafio pode ser baixada do site oficial:
http://www.openwall.com/john/

Introdução:

Baseado na aula de Administração de Usuários, nós como Administradores sempre temos que estar atentos com as senhas dos usuários, afinal, elas poderão ser as portas para uma intrusão no sistema.

Objetivo:

Este desafio tem como objetivo, descobrir as possíveis senhas fracas que possam vir a existir no sistema, usando um programa chamado John the Ripper, que tem a função de tentar quebrar senhas do sistema baseado no arquivo de senhas (/etc/shadow)

Aqui está um trecho do meu arquivo /etc/shadow, que é responsável por armazenar as senhas criptografadas dos usuários do sistema.

====================== Arquivo /etc/shadow =======================
root:$1$YtJpdTAX$9FzIngB/wriqze2vl8SH3.:14162:0:99999:7:::
user:$1$daE7WJYF$bHcNTBPNupjvoMQRWlvy/1:14162:0:99999:7:::


=============================================================

O objetivo é conseguir a senha do ROOT e do Usuário USER.

O aluno que completar o desafio, deverá enviar um arquivo (formatos aceitos: odt, txt, pdf), com os passos feitos desde instalação até a descoberta da senha para que possa ser analisado.
Não serão recebidos arquivos por e-mail.

Quem será o primeiro???
